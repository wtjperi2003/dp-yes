##### 范数

                    <p style="text-align:center;"><img alt="" class="has" height="465" src="https://img-blog.csdnimg.cn/20190702140900306.png?,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly94aW9uZ3lpbWluZy5ibG9nLmNzZG4ubmV0,size_16,color_FFFFFF,t_70" width="986"></p> 
<p style="text-indent:0;">&nbsp; &nbsp;阅读文献时，经常看到各种范数，机器学习中的稀疏模型等，也有各种范数，其名称往往容易混淆，例如：L1范数也常称为“1-范数”，但又和真正的1-范数又有很大区别。下面将依次介绍各种范数。</p> 
<h2><a name="t0"></a><a name="t0"></a><strong>1、向量的范数</strong></h2> 
<p><strong>&nbsp; 向量的1-范数： <img alt="{\left\| X \right\|_1} = \sum\limits_{i = 1}^n {\left| {​{x_i}} \right|}" class="mathcode" src="https://private.codecogs.com/gif.latex?%5Cinline%20%7B%5Cleft%5C%7C%20X%20%5Cright%5C%7C_1%7D%20%3D%20%5Csum%5Climits_%7Bi%20%3D%201%7D%5En%20%7B%5Cleft%7C%20%7B%7Bx_i%7D%7D%20%5Cright%7C%7D">&nbsp;;&nbsp;</strong>各个元素的绝对值之和；</p> 
<p style="margin-left:0cm;"><strong>&nbsp; 向量的2-范数：<img alt="{\left\| X \right\|_2} = {\left( {\sum\limits_{i = 1}^n {​{x_i}^2} } \right)^{\frac{1}{2}}} = \sqrt {\sum\limits_{i = 1}^n {​{x_i}^2} }" class="mathcode" src="https://private.codecogs.com/gif.latex?%5Cinline%20%7B%5Cleft%5C%7C%20X%20%5Cright%5C%7C_2%7D%20%3D%20%7B%5Cleft%28%20%7B%5Csum%5Climits_%7Bi%20%3D%201%7D%5En%20%7B%7Bx_i%7D%5E2%7D%20%7D%20%5Cright%29%5E%7B%5Cfrac%7B1%7D%7B2%7D%7D%7D%20%3D%20%5Csqrt%20%7B%5Csum%5Climits_%7Bi%20%3D%201%7D%5En%20%7B%7Bx_i%7D%5E2%7D%20%7D"></strong>；每个元素的平方和再开平方根；</p> 
<p style="margin-left:0cm;"><strong>&nbsp; 向量的无穷范数：</strong><img alt="{\left\| X \right\|_\infty } = \mathop {\max }\limits_{1 \le i \le n} \left| {​{x_i}} \right|" class="mathcode" src="https://private.codecogs.com/gif.latex?%5Cinline%20%7B%5Cleft%5C%7C%20X%20%5Cright%5C%7C_%5Cinfty%20%7D%20%3D%20%5Cmathop%20%7B%5Cmax%20%7D%5Climits_%7B1%20%5Cle%20i%20%5Cle%20n%7D%20%5Cleft%7C%20%7B%7Bx_i%7D%7D%20%5Cright%7C"></p> 
<p style="margin-left:0cm;"><strong><em>&nbsp; p</em>-范数：</strong><img alt="{\left\| X \right\|_p} = {\left( {\sum\limits_{i = 1}^n {​{​{\left| {​{x_i}} \right|}^p}} } \right)^{\frac{1}{p}}}" class="mathcode" src="https://private.codecogs.com/gif.latex?%7B%5Cleft%5C%7C%20X%20%5Cright%5C%7C_p%7D%20%3D%20%7B%5Cleft%28%20%7B%5Csum%5Climits_%7Bi%20%3D%201%7D%5En%20%7B%7B%7B%5Cleft%7C%20%7B%7Bx_i%7D%7D%20%5Cright%7C%7D%5Ep%7D%7D%20%7D%20%5Cright%29%5E%7B%5Cfrac%7B1%7D%7Bp%7D%7D%7D">，其中正整数<em>p</em>≥1，并且有<img alt="\mathop {\lim }\limits_{p \to \infty } {\left\| X \right\|_p} = \mathop {\max }\limits_{1 \le i \le n} \left| {​{x_i}} \right|" class="mathcode" src="https://private.codecogs.com/gif.latex?%5Cmathop%20%7B%5Clim%20%7D%5Climits_%7Bp%20%5Cto%20%5Cinfty%20%7D%20%7B%5Cleft%5C%7C%20X%20%5Cright%5C%7C_p%7D%20%3D%20%5Cmathop%20%7B%5Cmax%20%7D%5Climits_%7B1%20%5Cle%20i%20%5Cle%20n%7D%20%5Cleft%7C%20%7B%7Bx_i%7D%7D%20%5Cright%7C"></p> 
<p style="margin-left:0cm;">&nbsp;</p> 
<p style="margin-left:0cm;"><span style="color:#3399ea;">例：向量<em>X</em>=[2, 3, -5, -7] ，求向量的1-范数，2-范数和无穷范数。</span></p> 
<p style="margin-left:0cm;"><span style="color:#3399ea;">向量的1-范数：</span>各个元素的绝对值之和；<img alt="{\left\| X \right\|_1}" class="mathcode" src="https://private.codecogs.com/gif.latex?%7B%5Cleft%5C%7C%20X%20%5Cright%5C%7C_1%7D">=2+3+5+7=17；</p> 
<p style="margin-left:0cm;">Matlab代码：<em>X</em>=[2, 3, -5, -7]; XLfs1=norm(<em>X</em>,1);</p> 
<p style="margin-left:0cm;">&nbsp;</p> 
<p style="margin-left:0cm;"><span style="color:#3399ea;">向量的2-范数：</span>每个元素的平方和再开平方根；<img alt="{\left\| X \right\|_2} = {\left( {​{\rm{2}} \times {\rm{2}} + {\rm{3}} \times {\rm{3}} + {\rm{5}} \times {\rm{5}} + {\rm{7}} \times {\rm{7}}} \right)^{\frac{1}{2}}} = 9.3274" class="mathcode" src="https://private.codecogs.com/gif.latex?%7B%5Cleft%5C%7C%20X%20%5Cright%5C%7C_2%7D%20%3D%20%7B%5Cleft%28%20%7B%7B%5Crm%7B2%7D%7D%20%5Ctimes%20%7B%5Crm%7B2%7D%7D%20&amp;plus;%20%7B%5Crm%7B3%7D%7D%20%5Ctimes%20%7B%5Crm%7B3%7D%7D%20&amp;plus;%20%7B%5Crm%7B5%7D%7D%20%5Ctimes%20%7B%5Crm%7B5%7D%7D%20&amp;plus;%20%7B%5Crm%7B7%7D%7D%20%5Ctimes%20%7B%5Crm%7B7%7D%7D%7D%20%5Cright%29%5E%7B%5Cfrac%7B1%7D%7B2%7D%7D%7D%20%3D%209.3274">；</p> 
<p style="margin-left:0cm;">Matlab代码：<em>X</em>=[2, 3, -5, -7]; XLfs2=norm(X,2);</p> 
<p style="margin-left:0cm;">&nbsp;</p> 
<p style="margin-left:0cm;"><span style="color:#3399ea;">向量的无穷范数：</span></p> 
<p style="margin-left:0cm;"><span style="color:#3399ea;">（1）正无穷范数：</span>向量的所有元素的绝对值中<strong>最大</strong>的；即<em>X</em>的正无穷范数为：7；</p> 
<p style="margin-left:0cm;">Matlab代码：<em>X</em>=[2, 3, -5, -7]; XLfsz=norm(<em>X</em>,inf);</p> 
<p style="margin-left:0cm;">&nbsp;</p> 
<p style="margin-left:0cm;"><span style="color:#3399ea;">（2）负无穷范数：</span>向量的所有元素的绝对值中<strong>最小</strong>的；即<em>X</em>的负无穷范数为：2；</p> 
<p style="margin-left:0cm;">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; Matlab代码：<em>X</em>=[2, 3, -5, -7]; XLfsf=norm(<em>X</em>,-inf);</p> 
<p style="margin-left:0cm;">&nbsp;</p> 
<h2><a name="t1"></a><a name="t1"></a><strong>2、矩阵的范数</strong></h2> 
<p style="margin-left:0cm;">设：向量<img alt="X \in {R^n}" class="mathcode" src="https://private.codecogs.com/gif.latex?X%20%5Cin%20%7BR%5En%7D">，矩阵<img alt="A \in {R^{n \times n}}" class="mathcode" src="https://private.codecogs.com/gif.latex?A%20%5Cin%20%7BR%5E%7Bn%20%5Ctimes%20n%7D%7D">，例如矩阵<em>A</em>为：</p> 
<p style="margin-left:0cm;"><em>A</em>=[2, 3, -5, -7;</p> 
<p style="margin-left:0cm;">&nbsp;&nbsp; 4, 6,&nbsp; 8, -4;</p> 
<p style="margin-left:0cm;">&nbsp;&nbsp; 6, -11, -3, 16];</p> 
<p style="margin-left:0cm;"><strong>（1）矩阵的1-范数（列模）：</strong><img alt="{\left\| A \right\|_1} = \mathop {\max }\limits_{X \ne 0} \frac{​{​{​{\left\| {AX} \right\|}_1}}}{​{​{​{\left\| X \right\|}_1}}} = \mathop {\max }\limits_{1 \le j \le n} \sum\limits_{i = 1}^n {\left| {​{a_{ij}}} \right|}" class="mathcode" src="https://private.codecogs.com/gif.latex?%7B%5Cleft%5C%7C%20A%20%5Cright%5C%7C_1%7D%20%3D%20%5Cmathop%20%7B%5Cmax%20%7D%5Climits_%7BX%20%5Cne%200%7D%20%5Cfrac%7B%7B%7B%7B%5Cleft%5C%7C%20%7BAX%7D%20%5Cright%5C%7C%7D_1%7D%7D%7D%7B%7B%7B%7B%5Cleft%5C%7C%20X%20%5Cright%5C%7C%7D_1%7D%7D%7D%20%3D%20%5Cmathop%20%7B%5Cmax%20%7D%5Climits_%7B1%20%5Cle%20j%20%5Cle%20n%7D%20%5Csum%5Climits_%7Bi%20%3D%201%7D%5En%20%7B%5Cleft%7C%20%7B%7Ba_%7Bij%7D%7D%7D%20%5Cright%7C%7D">；矩阵的每一列上的元素绝对值先求和，再从中取个最大的，（列和最大）；即矩阵<em>A</em>的1-范数为：27</p> 
<p style="margin-left:0cm;">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; Matlab代码：fs1=norm(A,1);</p> 
<p style="margin-left:0cm;">&nbsp;</p> 
<p style="margin-left:0cm;"><strong>（2）矩阵的2-范数（谱模）：</strong><img alt="{\left\| A \right\|_2} = \mathop {\max }\limits_{X \ne 0} \frac{​{​{​{\left\| {AX} \right\|}_2}}}{​{​{​{\left\| X \right\|}_2}}} = \sqrt {​{\lambda _{\max }}({A^T}A)} = \sqrt {\mathop {\max }\limits_{1 \le i \le n} \left| {​{\lambda _i}} \right|}" class="mathcode" src="https://private.codecogs.com/gif.latex?%7B%5Cleft%5C%7C%20A%20%5Cright%5C%7C_2%7D%20%3D%20%5Cmathop%20%7B%5Cmax%20%7D%5Climits_%7BX%20%5Cne%200%7D%20%5Cfrac%7B%7B%7B%7B%5Cleft%5C%7C%20%7BAX%7D%20%5Cright%5C%7C%7D_2%7D%7D%7D%7B%7B%7B%7B%5Cleft%5C%7C%20X%20%5Cright%5C%7C%7D_2%7D%7D%7D%20%3D%20%5Csqrt%20%7B%7B%5Clambda%20_%7B%5Cmax%20%7D%7D%28%7BA%5ET%7DA%29%7D%20%3D%20%5Csqrt%20%7B%5Cmathop%20%7B%5Cmax%20%7D%5Climits_%7B1%20%5Cle%20i%20%5Cle%20n%7D%20%5Cleft%7C%20%7B%7B%5Clambda%20_i%7D%7D%20%5Cright%7C%7D">，其中&nbsp;&nbsp; <img alt="{\lambda _i}" class="mathcode" src="https://private.codecogs.com/gif.latex?%7B%5Clambda%20_i%7D">为<img alt="{A^T}A" class="mathcode" src="https://private.codecogs.com/gif.latex?%7BA%5ET%7DA">的特征值；矩阵<img alt="" class="has" height="32" src="https://img-blog.csdn.net/20180814201555453?watermark/2/text/aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L3phaXNodWl5aWZhbmd4eW0=/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70" width="43">的最大特征值开平方根。</p> 
<p style="margin-left:0cm;">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; Matlab代码：fs2=norm(A,2);</p> 
<p style="margin-left:0cm;">&nbsp;</p> 
<p style="margin-left:0cm;"><strong>（3）矩阵的无穷范数（行模）：</strong><img alt="{\left\| A \right\|_\infty } = \mathop {\max }\limits_{X \ne 0} \frac{​{​{​{\left\| {AX} \right\|}_\infty }}}{​{​{​{\left\| X \right\|}_\infty }}} = \mathop {\max }\limits_{1 \le {\rm{i}} \le n} \sum\limits_{j = 1}^n {\left| {​{a_{ij}}} \right|}" class="mathcode" src="https://private.codecogs.com/gif.latex?%7B%5Cleft%5C%7C%20A%20%5Cright%5C%7C_%5Cinfty%20%7D%20%3D%20%5Cmathop%20%7B%5Cmax%20%7D%5Climits_%7BX%20%5Cne%200%7D%20%5Cfrac%7B%7B%7B%7B%5Cleft%5C%7C%20%7BAX%7D%20%5Cright%5C%7C%7D_%5Cinfty%20%7D%7D%7D%7B%7B%7B%7B%5Cleft%5C%7C%20X%20%5Cright%5C%7C%7D_%5Cinfty%20%7D%7D%7D%20%3D%20%5Cmathop%20%7B%5Cmax%20%7D%5Climits_%7B1%20%5Cle%20%7B%5Crm%7Bi%7D%7D%20%5Cle%20n%7D%20%5Csum%5Climits_%7Bj%20%3D%201%7D%5En%20%7B%5Cleft%7C%20%7B%7Ba_%7Bij%7D%7D%7D%20%5Cright%7C%7D">；矩阵的每一行上的元素绝对值先求和，再从中取个最大的，（行和最大）</p> 
<p style="margin-left:0cm;">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp;Matlab代码：fswq=norm(A,inf);</p> 
<p style="margin-left:0cm;">&nbsp;</p> 
<p style="margin-left:0cm;">&nbsp; 下面要介绍关于机器学习中稀疏表示等一些地方用到的范数，一般有核范数，L0范数，L1范数（有时很多人也叫1范数，这就让初学者很容易混淆），L21范数（有时也叫2范数），F范数等，这些范数都是为了解决实际问题中的困难而提出的新的范数定义，不同于前面矩阵的范数。</p> 
<p style="margin-left:0cm;">关于核范数，L0范数，L1范数等解释见博客：</p> 
<p style="margin-left:0cm;"><a href="http://www.cnblogs.com/MengYan-LongYou/p/4050862.html">http://www.cnblogs.com/MengYan-LongYou/p/4050862.html</a></p> 
<p style="margin-left:0cm;"><a href="https://blog.csdn.net/u013066730/article/details/51145889">https://blog.csdn.net/u013066730/article/details/51145889</a></p> 
<p style="margin-left:0cm;"><a href="http://blog.sina.com.cn/s/blog_7103b28a0102w73g.html">http://blog.sina.com.cn/s/blog_7103b28a0102w73g.html</a></p> 
<p style="margin-left:0cm;">&nbsp;</p> 
<p style="margin-left:0cm;"><strong>（4）矩阵的核范数：</strong>矩阵的奇异值（将矩阵svd分解）之和，这个范数可以用来低秩表示（因为最小化核范数，相当于最小化矩阵的秩——低秩）；</p> 
<p style="margin-left:0cm;">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp;Matlab代码：JZhfs=sum(svd(A));</p> 
<p style="margin-left:0cm;">&nbsp;</p> 
<p style="margin-left:0cm;"><strong>（5）矩阵的L0范数：</strong>矩阵的非0元素的个数，通常用它来表示稀疏，L0范数越小0元素越多，也就越稀疏。</p> 
<p style="margin-left:0cm;">&nbsp;</p> 
<p style="margin-left:0cm;"><strong>（6）矩阵的L1范数：</strong>矩阵中的每个元素绝对值之和，它是L0范数的最优凸近似，因此它也可以近似表示稀疏；</p> 
<p style="margin-left:0cm;">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp;Matlab代码：JZL1fs=sum(sum(abs(A)));</p> 
<p style="margin-left:0cm;">&nbsp;</p> 
<p style="margin-left:0cm;"><strong>（7）矩阵的F范数：</strong>矩阵的各个元素平方之和再开平方根，它通常也叫做矩阵的L2范数，它的有点在它是一个凸函数，可以求导求解，易于计算；</p> 
<p style="margin-left:0cm;">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp;Matlab代码：JZFfs=norm(A,'fro');</p> 
<p style="margin-left:0cm;">&nbsp;</p> 
<p style="margin-left:0cm;"><strong>（8）矩阵的L21范数：</strong>矩阵先以每一列为单位，求每一列的F范数（也可认为是向量的2范数），然后再将得到的结果求L1范数（也可认为是向量的1范数），很容易看出它是介于L1和L2之间的一种范数</p> 
<p style="margin-left:0cm;">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; Matlab代码：JZL21fs=norm(A(:,1),2) + norm(A(:,2),2) + norm(A(:,3),2)++ norm(A(:,4),2);</p> 
<p style="margin-left:0cm;">&nbsp;</p>
